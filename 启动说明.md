# 芽目标本地版启动说明

这是“前端 + 后端 + 本地 MySQL 数据库”的版本。

## 打开方式

1. 确认 MySQL 已经启动。
2. 打开 PowerShell。
3. 输入：

```powershell
cd C:\Users\rl109\Desktop\芽目标
npm start
```

4. 浏览器打开：

```text
http://localhost:3001
```

不要再用双击 `index.html` 的方式打开新版，因为新版需要连接后端和数据库。

## 成功标志

看到以下内容，就说明启动成功：

- 页面能打开“芽目标”
- 首页能看到目标列表
- 可以创建目标
- 可以点击打卡
- 复盘可以保存
- 刷新页面后，数据还在

## 数据库说明

本地数据库名：

```text
ya_mubiao_demo
```

目前使用三张表：

- `goals`：目标
- `checkins`：打卡记录
- `reviews`：复盘记录

数据库账号密码写在 `.env` 文件里。这个文件包含隐私信息，不要发给别人。

## 排查步骤

### 1. 页面打不开

先确认有没有运行：

```powershell
npm start
```

然后打开：

```text
http://localhost:3001
```

### 2. 提示后端没启动

说明页面打开了，但服务没运行。重新执行：

```powershell
cd C:\Users\rl109\Desktop\芽目标
npm start
```

### 3. 提示数据库连接失败

检查 MySQL 是否启动，以及 `.env` 里的账号密码是否正确。

当前配置是：

```text
localhost:3306
root
```

密码在 `.env` 文件里。

### 4. 端口被占用

当前网页服务用的是：

```text
3001
```

如果 `3001` 被占用，可以改 `.env` 里的 `PORT`，同时告诉我，我会帮你把前端连接一起调整。

### 5. 图片不显示

确认 `designv2` 文件夹还在：

```text
C:\Users\rl109\Desktop\芽目标\designv2
```

图片素材都从这个文件夹读取。

## 安装成手机 App

当前版本是 PWA，也就是“可以安装到手机桌面的网页 App”。

### 在电脑上预览

打开：

```text
http://localhost:3001
```

### 在手机上打开

手机和电脑需要连接同一个 Wi-Fi。

1. 在电脑 PowerShell 输入：

```powershell
ipconfig
```

2. 找到类似下面的地址：

```text
IPv4 地址 . . . . . . . . . . . . : 192.168.x.x
```

3. 在手机浏览器打开：

```text
http://你的电脑IP:3001
```

示例：

```text
http://192.168.1.23:3001
```

### 添加到桌面

Android Chrome：

1. 打开网页
2. 点右上角菜单
3. 点“添加到主屏幕”或“安装应用”

iPhone Safari：

1. 打开网页
2. 点底部分享按钮
3. 点“添加到主屏幕”

### 成功标志

手机桌面出现“芽目标”图标，点开后是独立窗口，不像普通浏览器标签页。
